Install requirements to a conda env: pip install -r requirements.txt 
The models download in preceremony.
Python version: 3.10

Run gg_api.preceremony() followed by gg_api.main() to generate results.
Check autograder scores with python autograder.py  
Human readable output is in results.txt 
Autograder readable output is in results.json 

Additional goal: sentiment for presenters and winners
